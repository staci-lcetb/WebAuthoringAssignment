The Span word is red.

Tables & Caption

When inserting a table a caption is needed to indicate what you will be seeing in the table. The table is organised by using the table row, table header and table data. The row indicates which row of the table. Table header is a heading to that row and table data is the text or content put in the cell.

Example of the code

 <table width="200" border="5" bordercolor="#AB50B0" bgcolor="#F3BCE3" cellspacing="5" cellpadding="4" align="center"> <caption>This is a Table</caption> <tbody> <tr> <td>Header Here</td> <td>Content Here</td> </tr> <tr> <td>Header Here</td> <td>Content Here</td> </tr> <tr> <td>Header Here</td> <td>Content Here</td> </tr> </tbody> </table>

Working example

This is a Table
Header Here Content Here
Header Here Content Here
Header Here Content Here

Links

A HTML link is a hyperlink that can be internal or external. An internal link allows you to move through the pages of a website without leaving the site. An external link takes you off the website to a new page on a new tab.

Example of the code

<a href="second page.html">Page 2</a>

Working example

Page 2

Quote & Blockquote

A standard quote is used when you want a sentence to be made a quote. A blockquote is used when it is a direct quote from a source.

Example of the code

<q>"This is the day"</q>
<blockquote cite="https://eu.usatoday.com/story/news/nation/2019/01/21/martin-luther-king-jr-quotes-10-most-popular/2636024002/">"The time is always right to do what is right."
</blockquote>

Working example

"This is the day"
"The time is always right to do what is right."
